Alpine — это надёжный и простой инструмент для создания поведения непосредственно в вашей разметке. Думайте об этом как о jQuery для современного Интернета. Добавьте тег `script` и приступайте к работе.

```html wrap {1}
<script src="//unpkg.com/alpinejs" defer></script>

<div x-data="{ open: false }">
    <button @click="open = true">Развернуть</button>

    <span x-show="open">
        Контент...
    </span>
</div>
```

Alpine — это коллекция из **18 атрибутов**, **9 свойств** и **3 методов**.

Нет лучшего способа разобраться в том, что такое Alpine и на что он способен, чем увидеть это своими глазами:

      <Card title="<a href=/directives/show>x-show</a>">
        Переключайте видимость элемента:
        ```html wrap "x-show"
        <div x-show="open">
          ...
        </div>
        ```
      </Card>
      <Card title="<a href=/directives/bind>x-bind</a>">
        Устанавливайте динамические атрибуты HTML для элемента:
        ```html wrap "x-bind"
        <div x-bind:class="! open ? 'hidden' : ''">
          ...
        </div>
        ```
      </Card>
      <Card title="<a href=/directives/on>x-on</a>">
        Прослушивайте события браузера на элементе:
        ```html wrap "x-on"
        <button x-on:click="open = ! open">
          Переключить
        </button>
        ```
      </Card>
      <Card title="<a href=/directives/text>x-text</a>">
        Устанавливайте текстовое содержимое элемента:
        ```html wrap "x-text"
        <div>
          Авторское право © <span x-text="new Date().getFullYear()"></span>
        </div>
        ```
      </Card>
      <Card title="<a href=/directives/html>x-html</a>">
        Устанавливайте внутренний HTML элемента:
        ```html wrap "x-html"
        <div x-html="(await axios.get('/some/html/partial')).data">
          ...
        </div>
        ```
      </Card>
      <Card title="<a href=/directives/model>x-model</a>">
        Синхронизируйте часть данных с элементом ввода:
        ```html wrap "x-model"
        <div x-data="{ search: '' }">
          <input type="text" x-model="search">
          Поиск: <span x-text="search"></span>
        </div>
        ```
      </Card>
      <Card title="<a href=/directives/modelable>x-modelable</a>">
        Создавайте двустороннюю привязку данных для пользовательских компонентов:
        ```html wrap "x-modelable"
        <div x-data="{ value: '' }">
          <input x-modelable="value" type="text">
          <p>Value: <span x-text="value"></span></p>
        </div>
        ```
      </Card>
      <Card title="<a href=/directives/for>x-for</a>">
        Повторяйте блок HTML на основе набора данных:
        ```html wrap "x-for"
        <template x-for="post in posts">
          <h2 x-text="post.title"></h2>
        </template>
        ```
      </Card>
      <Card title="<a href=/directives/transition>x-transition</a>">
        Перемещайте элемент внутрь и наружу с помощью переходов CSS:
        ```html wrap "x-transition"
        <div x-show="open" x-transition>
          ...
        </div>
        ```
      </Card>
      <Card title="<a href=/directives/effect>x-effect</a>">
        Выполняйте скрипт каждый раз, когда меняется одна из его зависимостей:
        ```html wrap "x-effect"
        <div x-effect="console.log('count: '+count)"></div>
        ```
      </Card>
      <Card title="<a href=/directives/ignore>x-ignore</a>">
        Предотвращайте инициализацию блока HTML с помощью Alpine:
        ```html wrap "x-ignore"
        <div x-ignore>
          ...
        </div>
        ```
      </Card>
      <Card title="<a href=/directives/ref>x-ref</a>">
        Ссылайтесь на элементы напрямую по их указанным ключам, используя магическое свойство `$refs`:
        ```html wrap "x-ref"
        <input type="text" x-ref="content">
        <button x-on:click="navigator.clipboard.writeText($refs.content.value)">
          Скопировать
        </button>
        ```
      </Card>
      <Card title="<a href=/directives/cloak>x-cloak</a>">
        Скрывайте блок HTML до тех пор, пока Alpine не завершит инициализацию его содержимого:
        ```html wrap "x-cloak"
        <div x-cloak>
          ...
        </div>
        ```
      </Card>
      <Card title="<a href=/directives/teleport>x-teleport</a>">
        Перемещайте содержимое компонента в другое место в DOM, сохраняя его реактивность:
        ```html wrap "x-teleport"
        <div x-data="{ show: false }">
          <button x-on:click="show = !show">
            Переключить
          </button>
          <template x-if="show">
            <div x-teleport="#target">
              Это будет телепортировано!
            </div>
          </template>
        </div>
        <div id="target"></div>
        ```
      </Card>
      <Card title="<a href=/directives/if>x-if</a>">
        Добавляйте/удаляйте блок HTML по условию:
        ```html wrap "x-if"
        <template x-if="open">
          <div>...</div>
        </template>
        ```
      </Card>
      <Card title="<a href=/directives/id>x-id</a>">
        Генерируйте уникальные идентификаторы для элементов:
        ```html wrap "x-id"
        <div x-id="['text-input']">
          <label x-bind:for="$id('text-input')">Имя</label>
          <!-- for="text-input-1" -->

          <input x-bind:id="$id('text-input')" />
          <!-- id="text-input-1" -->
        </div>
        ```
      </Card>
    </CardGrid>
  </TabItem>
  <TabItem label="9 свойств">
    <CardGrid stagger>
      <Card title="<a href=/magics/el>$el</a>">
        Ссылайтесь на текущий элемент DOM:
        ```html wrap "$el"
        <div x-init="new Pikaday($el)"></div>
        ```
      </Card>
      <Card title="<a href=/magics/refs>$refs</a>">
        Ссылайтесь на элемент по ключу (задается с помощью `x-ref`):
        ```html wrap "$refs"
        <div x-init="$refs.button.remove()">
          <button x-ref="button">Удалите меня</button>
        </div>
        ```
      </Card>
      <Card title="<a href=/magics/store>$store</a>">
        Получайте доступ к глобальному хранилищу, зарегистрированному с помощью `Alpine.store(...)`:
        ```html wrap "$store"
        <h1 x-text="$store.site.title"></h1>
        ```
      </Card>
      <Card title="<a href=/magics/watch>$watch</a>">
        Отслеживайте фрагмент данных и при его изменении запускайте предоставленный обратный вызов:
        ```html wrap "$watch"
        <div x-init="$watch('count', value => {
          console.log('count: ' + value)
        })">...</div>
        ```
      </Card>
      <Card title="<a href=/magics/dispatch>$dispatch</a>">
        Отправляйте пользовательское событие браузера из текущего элемента:
        ```html wrap "$dispatch"
        <div x-on:notify="...">
          <button x-on:click="$dispatch('notify')">...</button>
        </div>
        ```
      </Card>
      <Card title="<a href=/magics/nexttick>$nextTick</a>">
        Дожидайтесь следующего «тика» (отрисовка в браузере) для запуска части кода:
        ```html wrap "$nextTick"
        <div
          x-text="count"
          x-text="$nextTick(() => {
            console.log('count: ' + $el.textContent)
          })"
        >...</div>
        ```
      </Card>
      <Card title="<a href=/magics/root>$root</a>">
        Получайте доступ к ближайшему элементу в дереве DOM, содержащему `x-data`:
        ```html wrap "$root"
        <div x-data data-message="Привет, мир!">
          <button x-on:click="alert($root.dataset.message)">Привет!</button>
        </div>
        ```
      </Card>
      <Card title="<a href=/magics/data>$data</a>">
        Получайте доступ к реактивным данным компонента для использования в шаблонах и методах:
        ```html wrap "$data"
        <div x-data="{ count: 0 }">
          <button x-on:click="count++">Увеличить</button>
          <p>Счётчик: <span x-text="$data.count"></span></p>
        </div>
        ```

      </Card>
      <Card title="<a href=/magics/id>$id</a>">
        Генерируйте уникальный идентификатор для каждого экземпляра компонента:
        ```html wrap "$id"
        <div x-data="{ id: $id('text-input') }">
          <label x-bind:for="id">Какой-то текст... </label> <!-- "text-input-1" -->
          <input type="text" x-bind:id="id"> <!-- "text-input-1" -->
        </div>
        ```

    </CardGrid>

        ```html wrap "x-data" "Alpine.data"

        </div>
        ```

        ```html wrap "$store" "Alpine.store"
        <button x-on:click="$store.notifications.notify('...')">

        </button>
        ```
        ```js wrap "Alpine.store"
        Alpine.store('notifications', {
          items: [],

          notify(message) {
            this.items.push(message)
          }
        })
        ```
      </Card>
      <Card title="<a href=/globals/alpine-bind>Alpine.bind</a>">
        Повторно используйте объект атрибутов и привязывайтесь к нему с помощью `x-bind`:
        ```html wrap "x-bind" "Alpine.bind"
        <button x-bind="SomeButton"></button>
        ```
        ```js wrap "Alpine.bind"
        Alpine.bind('SomeButton', () => ({
          type: 'button',

          '@click'() {
            this.doSomething();
          },

          ':disabled'() {
            return this.shouldDisable;
          },
        }))
        ```
      </Card>
    </CardGrid>
  </TabItem>
